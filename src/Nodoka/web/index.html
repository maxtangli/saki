説明

完成度
・Rule 95%, UI 10%, network 10%。
プレイの流れ
・ページを開く時、自動的にWebSocketサーバーに接続し、ゲーム状態のJSONが返る。
・実行できる命令はボタンより表示されている。
・ボタンをクリックすると、命令がサーバーに送信され、ゲーム状態が更新される。
マルチプレイ
・ページを開く時、座席（東南西北）が分配される。
・複数のページを開いたら、マルチプレイを体験できる。
・空いている座席はAIが担当され、現状はランダム打ちとなる。
